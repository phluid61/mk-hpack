Documentation Index

This directory contains the full documentation set for mkhpack, a C library implementing HPACK header compression for HTTP/2 (RFC 7541).

Documentation Inventory

DEPLOYMENT.md Build process, CI/CD pipelines, packaging, and installation

Quick Navigation by Role

  • New contributor: Start with DEVELOPER_GUIDE.md, then ARCHITECTURE.md
  • Library consumer: Start with API.md, then the usage examples in the README
  • Packager / release engineer: Start with DEPLOYMENT.md, then CONFIGURATION.md
  • Debugging a problem: Start with TROUBLESHOOTING.md
